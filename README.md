## Что сделано по модулям

### Модуль 1 — проектирование и разработка

- Спроектировал БД: таблицы `users` и `applications`, связь один-ко-многим
- Схема в `database/init.sql` (ER-диаграмма отдельно в draw.io)
- Регистрация: логин от 6 символов (латиница), пароль от 8, ФИО, телефон, email
- Вход / выход, JWT-токен
- Личный кабинет, список своих заявок
- Форма заявки: помещение, дата `ДД.ММ.ГГГГ`, способ оплаты (QR, МИР, постоплата)
- Админ-панель: логин `Admin26`, пароль `Demo20`
- Статусы заявок: «Новая» → «Банкет назначен» / «Банкет завершен»
- Отзывы пользователя после смены статуса админом

### Модуль 2 — дизайн и UX

- Оформление под мобильный экран **390×844**
- Палитра из руководства по стилю (золото, персик, крем, красный, зелёный)
- Слайдер на главной: 4 фото помещений из ассетов, автопрокрутка 3 сек, стрелки
- Подсказки при ошибках в формах регистрации и входа
- Читаемый вывод заявок (не `hall`, а «Зал»)
- Админка: фильтр по статусу, поиск, сортировка, пагинация, модальное окно смены статуса

### Модуль 3 — доработка и полировка

- Доработал `init.sql`: индексы, триггеры обновления `updated_at`
- Блок «Качество сервиса» (`quality_img` из М3)
- Блок «Контакты» (адрес и телефон из улучшений ТЗ)
- Микроанимации: кнопки, появление карточек, пауза слайдера при наведении
- Комментарий администратора виден пользователю в «Мои заявки»

## Запуск

### 1. PostgreSQL

Создай базу (у меня `new1`) и скопируй настройки:

```bash
copy server\.env.example server\.env
```

Пример `server/.env`:

```
DB_HOST=localhost
DB_PORT=5432
DB_NAME=new1
DB_USER=postgres
DB_PASSWORD=postgres
JWT_SECRET=любой_секретный_ключ
ADMIN_LOGIN=Admin26
ADMIN_PASSWORD=Demo20
```

### 2. Установка и старт

```bash
npm run install:all
npm run dev
```

- Сайт: http://localhost:3000  
- API: http://localhost:5000  

### 3. Вход

|------|-------|--------|

| Пользователь | свой после регистрации | свой |
